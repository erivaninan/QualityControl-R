Le test de Cox et Stuart est un test non paramétrique utilisé pour détecter une tendance monotone dans une série temporelle, ici les séries des quantités de monoxyde de carbonne libérées en 2023 et en 2024. Ce tets est basé sur un test du signe appliqué aux valeurs de la série prises de manière symétrique autour du centre.

Voici les hypothèses du test :

  • \(H_0\) : Pas de tendance monotone dans la série temporelle

  • \(H_1\) : Il existe une tendance monotone significative dans la série temporelle

L’idée est de découper la série \(X_1, X_2, X_3, ... , X_n\) en deux moitiés selon la parité des indices des valeurs et d’appliquer un test du signe sur la différence des valeurs.

tri_2023 <- data.frame(CO = tri_2023)  # conversion en dataframe
tri_2024 <- data.frame(CO = tri_2024)

# Test de Cox et Stuart 
test_cox_stuart <- function(serie) {
  serie <- na.omit(serie)  # supprime les valeurs manquantes
  taille <- length(serie)

  # on ajuste pour les séries de taille impaire : on supprime le dernier élément
  if (taille %% 2 == 1) {
    serie <- serie[-taille]  
    taille <- length(serie)
  }

  k <- taille / 2  # moitié de la série
  premiere_moitie <- serie[1:k]  # première moitié
  deuxieme_moitie <- serie[(k+1):taille]  # deuxième moitié

  # Calcul des différences entre les deux moitiés
  differences <- deuxieme_moitie - premiere_moitie

  # Comptage des signes positifs et négatifs
  nb_positifs <- sum(differences > 0, na.rm=TRUE)
  nb_negatifs <- sum(differences < 0, na.rm=TRUE)

  # Application du test binomial sous H0 (répartition aléatoire des signes)
  test_signe <- binom.test(nb_positifs, nb_positifs + nb_negatifs, p=0.5, alternative="two.sided")

  return(test_signe)
}
